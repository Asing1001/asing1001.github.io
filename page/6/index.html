如何找 Windows cmd 輸入的 command 實體路徑？

Posted on 2016-08-08 | Edited on 2023-07-02 |

語法：Where + Command
範例：where node
輸出：C:\Program Files\nodejs\node.exe

會特別找這個是因為我常更新某個command後，卻發現版本依然停在舊的，這時候用where就知道大概是哪個環境變數和舊版程式在搞鬼囉！
