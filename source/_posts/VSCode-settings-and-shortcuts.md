- `command 0` - Focus on sidebar
- `cmd+d` - For don't save
- `gb` - For multi cursor in vim mode
- `cmd+shift+.` - show breadcumb

## Settings.json

```json
{
  "editor.autoIndent": true,
  "editor.snippetSuggestions": "top",
}
```
